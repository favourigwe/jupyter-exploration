# 📖 Project Overview

The NewsBot Intelligence System is a Natural Language Processing (NLP) project that analyzes and classifies news articles from the HuffPost News Category Dataset. The goal of the project is to demonstrate how NLP techniques can transform unstructured text into meaningful information that supports business intelligence and decision-making.

The system performs several key NLP tasks, including text preprocessing, feature extraction, sentiment analysis, news classification, and named entity recognition. These techniques allow the NewsBot to identify important information within news articles, classify them into categories, analyze their emotional tone, and present the results through clear visualizations.

---

# 🎯 Project Objectives

* Prepare and clean raw news article data.
* Extract meaningful text features using TF-IDF.
* Analyze sentence structure through Part-of-Speech tagging and dependency parsing.
* Perform sentiment analysis using TextBlob.
* Classify news articles into major news categories.
* Extract named entities such as people, organizations, and locations.
* Present findings through visualizations and business intelligence insights.

# 📂 Dataset

This project uses the **HuffPost News Category Dataset**, which contains more than 200,000 news articles collected from HuffPost across dozens of news categories, including Politics, Business, Entertainment, Technology, Sports, Wellness, and many others.

For this project, a sample of the dataset was processed to improve computational efficiency while preserving category diversity.

---

# 🚀 NLP Pipeline

The NewsBot follows the complete NLP workflow:

1. Dataset Preparation
2. Text Cleaning and Preprocessing
3. TF-IDF Feature Extraction
4. Part-of-Speech (POS) Tagging
5. Dependency Parsing
6. Sentiment Analysis
7. News Article Classification
8. Named Entity Recognition (NER)
9. Business Intelligence Visualizations

---
